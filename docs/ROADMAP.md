# Roadmap

This document outlines the high-level goals and future direction for the project.

## Current Focus (MVP)
- Solidifying the offline-first sync engine.
- Establishing robust data structures for grouped cards and parsed entries.
- Cloud Authentication and database integration.

## Next Phase
- Implementation of the RADAR Agent.
- Advanced conflict resolution strategies.
- Reminder systems.

## Long Term
- Integrations with third-party services (HEO, Google Calendar, Google Photos).
- Collaborative features (Family Sync).
- Multimodal capture (Voice).

*For the granular status of the current implementation, see [MVP_STATUS.md](./MVP_STATUS.md).*
